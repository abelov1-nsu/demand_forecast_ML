# 🛒 Система прог прогнозирования спроса

Учебный проект по созданию **системы прогнозирования спроса** для розничной сети магазинов  
с использованием **LSTM** (PyTorch) и удобным веб-интерфейсом на **Streamlit**.

## ✨ О проекте

Проект реализует end-to-end решение для прогнозирования ежедневных продаж товаров на основе:

- исторических данных о продажах
- цен и промо-активности
- физических характеристик товара
- сезонных колебаний

### Основные возможности

- Обучение современной рекуррентной модели **LSTM**
- Сравнение с простым baseline (скользящее среднее)
- Метрика качества — **MAE**
- Удобный веб-интерфейс на **Streamlit**:
  - Графики исторических продаж
  - Прогноз на произвольное количество дней вперед
  - Расчёт рекомендованного объёма закупки
  - Ключевые статистики по товару

## 📊 Данные и признаки

Используемые фичи для прогнозирования:

| Признак                  | Описание                              | Тип данных    |
|--------------------------|---------------------------------------|---------------|
| Исторические продажи     | Продажи за предыдущие дни             | Количественный|
| Цена                     | Обычная цена                          | Количественный|
| Цена со скидкой          | Цена после промо                      | Количественный|
| Промо-активность         | Флаг / сила промоакции                | Бинарный      |
| Размер                   | Габариты товара                       | Количественный|
| Вес                      | Вес единицы товара                    | Количественный|
| Кол-во в упаковке        | Штук в транспортной упаковке          | Количественный|
| Сезонность (sin)         | `sin(2π · day_of_year / 365)`         | Циклический   |
| Сезонность (cos)         | `cos(2π · day_of_year / 365)`         | Циклический   |

## 🧠 Модель

**Основная архитектура**  
→ **LSTM** (PyTorch)

- Длина входной последовательности: **120 дней**
- Функция потерь: **MSE**
- Оптимизатор: **Adam**
- Масштабирование: **MinMaxScaler** (отдельно для продаж и фич)

**Бейзлайн**  
→ Скользящее среднее (Moving Average)

**Метрика оценки**  
→ **MAE** (Mean Absolute Error)

### Результаты (пример)

| Модель               | MAE     |
|----------------------|---------|
| Скользящее среднее   | ~ 4.2   |
| LSTM                 | ~ 2.5   |
