Система прогнозирования спроса
Учебный проект по построению системы прогнозирования спроса для сети магазинов с использованием LSTM (PyTorch).

Описание проекта:
Проект реализует модель прогнозирования продаж на основе временных рядов и товарных характеристик.
Система включает:
- обучение модели LSTM;
- сравнение с базовой моделью (скользящее среднее);
- расчёт метрики MAE;
- веб-интерфейс на Streamlit для визуализации и прогноза.

Используемые данные
Для построения прогноза используются следующие признаки:
- Исторические продажи
- Цена
- Цена со скидкой
- Промо-активность
- Размер
- Вес
- Количество в упаковке

Сезонность (sin/cos кодирование дня года)
Сезонность кодируется следующим образом:
sin_day = sin(2π * day_of_year / 365)
cos_day = cos(2π * day_of_year / 365)

Модель
Основная модель:
LSTM (PyTorch)

Вход: последовательность из 120 дней
Признаки: масштабированные продажи + числовые характеристики + сезонность
Функция потерь: MSE
Оптимизатор: Adam

Baseline
Базовая модель — скользящее среднее (Moving Average).

Метрика
Для оценки качества используется:
MAE (Mean Absolute Error)

Установка:
pip install -r requirements.txt

Генерация данных
python generate_data.py

Обучение модели
python train.py

В процессе обучения:
- выполняется разделение на train/test;
- рассчитывается MAE;
- модель и скейлеры сохраняются в папку model/.

Запуск интерфейса
python -m streamlit run app.py

В интерфейсе доступны:
- визуализация исторических продаж;
- прогноз на выбранное количество дней;
- расчёт рекомендованного объёма закупки;
- базовая статистика по товару.
