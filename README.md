**Учебный проект по прогнозированию продаж товаров с использованием нейронной сети LSTM.**

**В проекте реализован полный pipeline:**
- генерация синтетических данных о продажах
- обучение модели LSTM
- оценка качества модели
- сохранение обученной модели
- веб-интерфейс на Streamlit для демонстрации прогноза

Проект демонстрирует базовый подход к решению задачи прогнозирования временных рядов в контексте розничных продаж.

**Генерация данных**
Данные генерируются синтетически и включают:
- идентификатор товара
- дату
- цену
- цену со скидкой
- наличие акции
- размер
- вес
- количество в упаковке
- продажи

**В данных моделируются:**
- сезонность (синусоидальная компонента)
- тренд
- влияние характеристик товара
- случайный шум
- выбросы

Продажи являются неотрицательными величинами.


Числовые признаки масштабируются с помощью StandardScaler.
**Модель**

Используется LSTM-сеть:
- 1 слой LSTM
- 64 скрытых нейрона
- линейный выходной слой
- функция потерь: MSE
- оптимизатор: Adam
- Модель обучается на последовательностях длиной 120 дней.

Обратное преобразование выполняется через:
- sales=exp⁡(y)−1
- sales=exp(y)−1

**Оценка качества**
Используются метрики:
- MAE (Mean Absolute Error)
- RMSE (Root Mean Squared Error)
- MAPE (Mean Absolute Percentage Error)

**Также реализован baseline-прогноз:**
продажи следующего дня равны продажам предыдущего дня.
Модель сравнивается с baseline.

**Установка**
Создать виртуальное окружение и установить зависимости:
pip install -r requirements.txt

**Зависимости**
requirements.txt:
- faker
- torch
- pandas
- numpy
- scikit-learn
- matplotlib
- streamlit
- joblib

**Обучение модели**
Запуск обучения:
python train.py
После обучения модель сохраняется в папку model/.

**Запуск веб-интерфейса**
streamlit run app.py

**Интерфейс позволяет:**
- выбрать товар
- построить прогноз
- визуализировать фактические и предсказанные значения
